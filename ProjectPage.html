Paintball: This game is 1st person pov, where the player is able to shoot at the enemies, racking up points by destroying them

Spaceshooter: This is a 2D game where the user is flying a spaceship, by trying to evade alien attacks to destroy them.

Haunted Jaunt: The 3rd person user game incorporates a haunted house full of ghost, where the player's task is to avoid any contact with the ghosts and try to get to the exit unharmed.

Aircraft Augmented: This was the first version of my research project, where I implemented image tracking for multiple aircratfs with the Hiro marker, using Vuforia SDK.

VR Cornhole: Created a VR version of the game cornhole. In this game the user can pick up beannags and try to score 21 points, as well as shift the board closer or further away for difficulty mode.

Mend: Collaborated with an industry company in a challenge posed by them to create a communication system between VR and pc characters. This work was done within the 12 week intensive bootcamp @XRTerra

Model Targets: This is a simple AR experience to learn Model Target tracking with various objects such as Coffee Mug, PS4 Dualshock controller & Oculus remote.
